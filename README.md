# Kaiko Options Strategy Backtester

Professional-grade cryptocurrency options analysis and backtesting platform powered by Kaiko market data.

![Python](https://img.shields.io/badge/python-3.11+-blue.svg)
![Streamlit](https://img.shields.io/badge/streamlit-1.31.0-red.svg)

## 🎯 Features

### Trading Strategies
- **Simple Volatility** - Trade based on IV percentiles
- **Skew Trading** - Exploit put/call IV differentials
- **Gamma Scalping** - Target high gamma for delta-neutral profits
- **Calendar Spreads** - Trade term structure inefficiencies
- **Straddle Screener** - Identify optimal entry points

### Advanced Analytics
- 📊 **3D IV Surface Visualization**
- 🔥 **Gamma Exposure (GEX) Profiles**
- 📈 **Greeks Dashboard** (Delta, Gamma, Vega, Theta)
- 🎯 **Skew Term Structure Analysis**
- 💡 **Real-time Signal Generation**

### AI-Powered
- 🤖 **Natural Language Strategy Parser** - Describe strategies in plain English
- 🧠 **Conversational Backtesting** - Ask questions about market conditions
- ⚡ **Automated Signal Detection**

### Data Management
- 💾 **Historical Snapshot Storage** (SQLite)
- 🔄 **Daily Data Capture**
- 📉 **Time-Series Backtesting**
